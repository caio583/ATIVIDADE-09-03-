Este código cria 6 padrões diferentes de tabelas:

Matriz completa - Todos os campos preenchidos com *

Diagonal principal vazia - Apenas a diagonal principal fica vazia

Diagonal secundária vazia - Apenas a diagonal secundária fica vazia

X vazio - Ambas as diagonais ficam vazias

Pirâmide - Triângulo inferior preenchido

Pirâmide invertida - Triângulo superior preenchido
